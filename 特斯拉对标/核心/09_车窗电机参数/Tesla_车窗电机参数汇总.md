# 特斯拉车窗电机参数与供应商

## 电机基本参数（基于LIN总线车窗电机标准）

| 参数 | 数值 | 备注 |
|------|------|------|
| **工作电压** | 12V DC | 汽车低压系统 |
| **额定功率** | ~100-150W | 典型车窗电机 |
| **额定电流** | ~8-12A | 正常升降 |
| **堵转电流** | ~25-35A | 触发防夹反转 |
| **通信协议** | LIN 2.x | 单线串行总线 |
| **LIN波特率** | 19.2 kbps | 标准LIN速率 |
| **位置反馈** | 霍尔传感器 | 脉冲计数测速 |
| **霍尔脉冲** | 若干/转 | 用于速度/位置检测 |
| **工作温度** | -40°C ~ +85°C | 汽车级 |

## 各国工厂供应商差异

### 美国 Fremont 工厂
- 车窗电机: **Mitsuba** (日本电装, 可能)
- 车窗调节器: **Brose** (德国, 可能)
- 玻璃: Saint-Gobain / Guardian (美国)

### 中国 Shanghai 工厂
- 玻璃: **Fuyao** (福耀玻璃) — 已确认
- 车窗系统供应商: 中国本地化 (具体待确认)
- >90%零件在中国采购（来源: AlsetEV / Gurufocus）

### 德国 Berlin 工厂
- 车窗系统: Brose (欧洲本地)
- 电机: 可能使用 Valeo / Mitsuba

## 结构特点
- 车窗电机与调节器作为一个总成出售（不可单独购买电机）
- 通信基于 LIN 总线，每侧门模块控制对应的车窗电机
- 电机内含霍尔传感器用于防夹检测
