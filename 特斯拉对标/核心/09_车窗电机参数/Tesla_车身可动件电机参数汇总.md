## 1. 车窗电机 (Window Motor)

特斯拉全系车型的车窗电机与调节器集成出售（不可单独购买电机），通过 LIN 总线与门模块通信。

### 1.1 通用电气参数

| 参数 | 典型值 | 备注 |
|------|--------|------|
| **工作电压** | 12V DC (9-16V) | 汽车低压系统 |
| **电机类型** | 永磁直流电机 (PMDC) | Brose / Mitsuba |
| **额定功率** | ~100-150W | 典型车窗升降 |
| **额定电流** | ~8-12A | 正常升降工况 |
| **堵转电流** | ~25-35A | 触发防夹反转的阈值 |
| **通信协议** | LIN 2.x | 单线串行总线 |
| **LIN波特率** | 19.2 kbps | 标准 LIN 速率 |
| **位置反馈** | 霍尔传感器 | 2-3个霍尔元件，脉冲计数 |
| **霍尔脉冲** | 若干脉冲/转 | 用于速度/位置/方向检测 |
| **减速比** | ~50:1 至 80:1 | 蜗轮蜗杆 + 行星齿轮 |
| **工作温度** | -40°C ~ +85°C | 汽车级 |
| **防护等级** | IP54 | 门板内部安装 |

### 1.4 LIN 总线通信

```
引脚定义(典型6针):
  Pin 1: 电源 (12V)
  Pin 2: 地 (GND)
  Pin 3: LIN 总线
  Pin 4-6: 霍尔传感器 A/B/C (或备用)
```

**LIN 信号内容**（来自开放数据库分析）:
- 主控: 门模块 (Door ECU / BCM)
- 从控: 车窗电机控制器
- 指令: 上升/下降/停止/标定
- 反馈: 当前位置、速度、堵转状态、温度

### 1.5 防夹相关电流参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 正常工作电流 | 5-12A | 匀速升降 |
| 遇到障碍物电流升高速率 | >100A/s | 快速上升触发防夹 |
| 防夹触发阈值 | 堵转电流的 70-80% | 约 20-28A |
| 软件限流保护 | ~35A | 硬件过流保护 |
| 防夹反转时间 | <50ms | 从检测到反转指令 |

## 附录A: CAN/LIN 信号相关

来自 commaai/opendbc 开源数据库的 Tesla 门控信号:

```
// Model 3/Y 车窗相关 DBC (部分)
// LIN 从控信号 - 门模块到车窗电机
// ID: 0x60 (左前窗)
// ID: 0x61 (右前窗)  
// ID: 0x62 (左后窗)
// ID: 0x63 (右后窗)

// 主要信号:
// LIN_CMD_WINDOW_UP: 1 byte 上升指令
// LIN_CMD_WINDOW_DOWN: 1 byte 下降指令
// LIN_CMD_WINDOW_STOP: 1 byte 停止
// LIN_STATUS_POSITION: 2 bytes 当前位置 (0-100%)
// LIN_STATUS_SPEED: 1 byte 当前速度
// LIN_STATUS_CURRENT: 1 byte 当前电流
// LIN_STATUS_TEMP: 1 byte 电机温度
```
