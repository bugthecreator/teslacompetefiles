Handle - Exterior Release - Door - Front - LH (Remove and Replace)

FRT No: 11451002

Removal

  1. Remove the exterior door handle bezel (refer to procedure).
  2. Remove the LH door glass (refer to procedure).
  3. Peel back the mucket seal then remove and discard the sealing patches (x3) from the door.

    View large image.

  4. Disconnect the door push sensor harness.

    View large image.

  5. Release the edge clip that secures the door push sensor harness to the inner belt.

    View large image.

  6. Remove the fasteners (x3) that secure the door handle to the door (torque 10 Nm).
  7. Manuever the door handle out from the door.

Installation

Installation procedure is the reverse of removal, except for the following:

  1. Replace the sealing patches.
  2. Ensure that the door handle is flush with the door and functions properly.
