Actuator - Latch - Hood (Remove and Replace)

FRT No: 11352202

Removal

  1. Use a trim tool to remove the tow hook cover.
    Note: Insert the trim tool into the notch at the top right portion of the tow hook cover.

    View large image.

  2. Release both hood latch manual release straps.
  3. Remove the front underhood storage unit (refer to procedure).
  4. Remove the bolts (x3) that secure the actuator bracket to the front end module (FEM) (torque 10 Nm).
    Note: Components have been removed in this graphic to aid clarity.

    View large image.

  5. Lift the actuator assembly above the FEM.
  6. Release the electrical connectors (x2) from the bottom of the actuator assembly.

    View large image.

  7. Release the edge clips (x3) that secure the hood latch cable to the FEM.
    Caution: Replace any broken clips.

    View large image.

  8. Release both hood latch cables (x2) from the latch assembly.

    View large image.

  9. Release the post clip from the 12V harness.
  10. Remove the actuator from the vehicle.

Installation

Installation procedure is the reverse of removal.
