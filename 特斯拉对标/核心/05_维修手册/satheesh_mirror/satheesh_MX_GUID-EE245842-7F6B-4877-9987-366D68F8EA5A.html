Cladding - Door - Rear - LH (Remove and Replace)

FRT No: 12204202

Special tool required for this procedure:

Supplier Part Number Description
Tesla 223-MR05029-TSL Ergonomic Silicone Seam Roller

Removal

  1. Remove the rear door closeout panel (refer to procedure).
  2. Disconnect the cladding sensor connector from the harness inside the door.

    View large image.

  3. Use a trim tool to release the clips (x4) that secure the cladding to the rear door.

    View large image.

  4. Remove the rear door cladding from the vehicle.

Installation

Installation procedure is the reverse of removal, except for the following:

  1. Clean the affected area on the vehicle with alcohol wipes.
  2. Apply new tape if reinstalling the cladding.
  3. Remove the tape backing.

    View large image.

  4. Install the rearmost clip first, then install the remaining clips (x3), working towards the front of the door.
    Caution: Failure to install the rearmost clip first can result in poor seating of the cladding.

    View large image.

  5. Press firmly on the cladding using a silicone seam roller to completely seal the cladding tape.
  6. Ensure proper operation of the rear door prior to reinstalling the rear door trim panel.
