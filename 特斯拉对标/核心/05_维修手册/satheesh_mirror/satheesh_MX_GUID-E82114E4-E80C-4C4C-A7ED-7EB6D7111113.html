Module - TPMS (Remove and Replace)

FRT No: 34043002

Removal

  1. Remove the headliner (refer to procedure).
  2. Disconnect the harness from the TPMS module.

    View large image.

  3. Slide the TPMS module toward the front of the vehicle to release it from the spine closing plate.
  4. Remove the TPMS module from the vehicle.

Installation

Installation procedure is the reverse of removal, except for the following:

  1. Perform the TPMS sensor ID learn procedure (refer to procedure).
  2. Perform a road test.
