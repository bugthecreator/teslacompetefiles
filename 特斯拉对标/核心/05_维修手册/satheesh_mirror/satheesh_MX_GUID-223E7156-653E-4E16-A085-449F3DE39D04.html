Cladding - Door - Front - LH (Remove and Replace)

FRT No: 12204002

Special tool required for this procedure:

Supplier Part Number Description
Tesla 223-MR05029-TSL Ergonomic Silicone Seam Roller

Removal

  1. Open the front door.
  2. Working from the rear, use a trim tool to release the clips (x6) and double sided tape that secure the cladding to the front door.
    Tip: After the rearmost clip is released, close the front door and carefully pull the cladding away from the door to release the remaining clips (x5).

    View large image.

  3. Remove any broken clips from the door.
  4. Remove the front door cladding from the vehicle.

Installation

Caution: Replace any broken clips.
  1. Clean any remaining double sided tape residue.
  2. Apply new tape if reinstalling the cladding. Remove the tape backing.

    View large image.

  3. Install the rearmost clip first then install the remaining clips (x5) working towards the front of the door.
    Caution: Failure to install the rearmost clip first can result in poor seating of the cladding.

    View large image.

  4. Press firmly on the cladding using a silicone seam roller to completely seal the double sided tape.
