    Pinch Sensor (Forward Edge) - Liftgate (Remove and Replace)

    Correction code 11358002

    Parent topic: 1135 - Latches, Strikers, and Pinch Sensors

    Removal

    1. Open the liftgate.
    2. Remove the pushpin clips (x3) that attach the LH and RH side pinch sensors to the liftgate.
    3. Disconnect the forward edge pinch sensor from the LH side pinch sensor assembly.
    4. Remove the forward edge liftgate pinch sensor from the rear lift gate glass.
      Note
      The forward edge pinch sensor is attached to the glass with double sided tape.

    Installation

    Installation procedure is the reverse of removal, except for the following:

    1. Wipe off all adhesive residue from the rear lift gate glass.
    2. Add some adhesive promoter to the rear upper lift gate glass area, and allow it to dry to the touch.
    3. Remove the tape backing from the pinch sensor and start attaching the sensor to the rear lift gate. Start on the right side moving to the left.
    4. Close the rear lift gate and depress the sensor to confirm it is working correctly.
      CAUTION
      Stop the lift gate with the other hand in case the new sensor is not working.
