    Module - Body Controller - BCM (Remove and Replace)

    Correction Code 17152302

    • 2023-07-21: Added Note about security-related toolbox actions.
    • 2023-07-10: Updated links.
    Note
    Some of the Toolbox Actions within this procedure are related to vehicle security, and can only be performed by Tesla employees or third parties authorized to perform security-related Toolbox Actions. To learn more, see the Service Subscription FAQs available in the user account dropdown menu.
    Warning
    If the 12V power supply is disconnected, do not attempt to open any doors with door glass in closed position. Failure to follow this instruction could result in door glass shatter.
    Note
    Before disconnecting the 12V power supply, ensure that the driver’s door window is fully open. Failure to follow this instruction could result in vehicle lockout.
    Parent topic: 1715 - Electronic Control Modules

    Removal

    1. Move the passenger seat fully rearward for access.
    2. Disconnect 12V power. See Disconnect 12V Power .
    3. Remove the RH center console closeout extension. See Assembly - Closeout Extension - LH - Center Console (Remove and Replace).
    4. Remove the passenger side dash footwell closing trim. See Assembly - Cover - Footwell - Passenger's (Remove and Replace).
    5. Carefully peel back the carpet for access to the Body Control Module (BCM) cover.
      CAUTION
      Take care not to damage component(s).
    6. Remove the bolts (x4) that secure the BCM cover to the mounting bracket (torque 5 Nm).
    7. Remove the BCM cover.
    8. Carefully release the BCM from the mount. Disconnect the harness connectors (x9) from the BCM.
      CAUTION
      Protect disconnected optical connectors from contamination and dirt ingress.
    9. Remove the BCM from the footwell.

    Installation

    Note
    Configuration changes from within Toolbox can only be performed by a Tesla employee at this time. Third parties must request the configuration change through their agreed Tesla technical support channel.
    Installation procedure is the reverse of removal, except for the following:
    • Reinstall the latest vehicle firmware, or update the firmware if not the latest. See Firmware Reinstall - Touchscreen or Firmware Update.
    • For the body controller, in Toolbox 3, click the Actions tab, type "PROC-BDY" into the search field, click PROC_BDY_X_PEKTRON-VIN-LEARNvia Toolbox: (link), click Run, and allow the routine to complete.
    • For the security controller, in Toolbox 3, click the Actions tab, type "PROC-SEC" into the search field, click PROC_SEC_X_PEKTRON-VIN-LEARNvia Toolbox: (link), click Run, and allow the routine to complete.
    • Relearn all keys (refer to procedure).
    • For early Model S vehicles, in Toolbox 3, click the Actions tab, type "immobilizer reset" into the search field, click PROC_DI_X_IMMOBILIZER-RESETvia Toolbox: (link), click Run, and allow the routine to complete.
